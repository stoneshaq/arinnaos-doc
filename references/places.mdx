# Places

Places are organisation-scoped reference records.

They can represent countries, regions, cities, neighbourhoods, areas, and pickup points.

## Hierarchy

Places can have parent and child places.

Example:

- Turkey
  - Cappadocia
    - Göreme
    - Ürgüp

## Use cases

Places can be linked to tour templates, sights, activities, departures, and operational planning.
