# Tour Departures

Tour departures are scheduled operating instances of tour template versions.

A departure links to a specific tour template version and stores a snapshot of the template at creation time.

## Departure data

Departures can manage:

- Departure number
- Start and end dates
- Capacity
- Definite status
- Guide assignments
- Transport assignments
- Accommodation records
- Departure services

## Template snapshots

The template snapshot stays stable even if the original tour template changes later.

## Operational assignments

ArinnaOS validates guide, transport, accommodation, and service links so they remain within the same organisation and compatible with the referenced template version.
