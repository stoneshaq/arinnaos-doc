# Passenger Visas

Passenger visas are stored separately from the master passenger record.

A visa record includes:

- Passenger
- Country
- Visa type
- Visa number
- Issue date
- Expiry date
- Entries
- Status
- Notes

Visa organisation scope must match the passenger organisation.
