# Tour Passengers

Tour passengers represent departure-specific operational state for a traveller.

Tour passenger data includes:

- Room type
- Room number
- Roommate
- Seat number
- Arrival flight
- Departure flight
- Tour notes
- Special requirements
- Joined and left dates

## Important boundary

Tour passenger data does not belong on the master passenger profile because it changes by departure.
