# Advisor Profiles

Advisor profiles represent internal or external sales/advisor identities.

Advisor profiles may be linked to users or used as standalone advisor records.

## Advisor relationships

Advisor profiles can be linked to:

- Customers
- Leads
- Passengers
- Quotes
- Bookings

## User linkage

Advisor profile user linkage must remain within the same organisation and active membership rules.
