# Bookings

Bookings are commercial contracts.

A booking is created explicitly from an accepted quote.

Quote acceptance alone does not create a booking.

## Conversion requirements

A quote can be converted only when:

- The quote status is accepted
- Accepted quote version exists
- Accepted quote scenario exists

## Snapshot preservation

Bookings copy commercial totals, accepted quote lineage, booking lines, and passenger snapshots.
