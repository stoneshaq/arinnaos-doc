# Support and Impersonation

Platform support can inspect organisation health and subscription state.

## Support summaries

Support summaries can include:

- Organisation state
- Subscription state
- Usage state
- Billing state

## Impersonation

Impersonation sessions are audited.

Current implementation records start and end events. Full session takeover is reserved for future workflow integration.
