# Quote Versions

A quote version freezes pricing at a point in time.

Issued or accepted versions should not be mutated.

## Repricing

If pricing changes, create a new version.

## Snapshot fields

Quote versions can store:

- Quote currency
- Base currency
- FX snapshot
- Traveller counts
- Passenger snapshot
- Totals
- Margin
- Scenario totals
- Snapshot metadata

## Acceptance

Accepted quotes must point to the accepted version and accepted scenario.
