# Roadmap

ArinnaOS already includes a broad backend foundation and the beginning of the UI foundation.

## Completed foundations

- Multi-tenant organisation model
- CRM
- Supplier network
- References
- Tours and departures
- Passengers
- Quotes
- Bookings and rooming
- Travel finance
- SaaS billing
- Platform administration
- Invite-only auth backend
- Scheduler and command infrastructure
- UI foundation
- Auth UI

## Upcoming product areas

- Platform admin UI
- Tenant shell refinement
- CRM UI
- Supplier UI
- Tours and operations UI
- Passenger UI
- Quotes and bookings UI
- Finance UI
- SaaS billing UI
- Documentation site
- Production readiness
