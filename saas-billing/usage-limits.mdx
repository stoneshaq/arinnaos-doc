# Usage Limits

Usage limits help enforce plan allowances.

## Counters

ArinnaOS tracks usage counters for resources such as:

- Users
- Customers
- Leads
- Passengers
- Suppliers
- Tour templates
- Departures
- Quotes
- Bookings
- Document storage

## Hard limits

Hard limits block actions.

## Soft limits

Soft limits allow temporary overflow and log warnings.
