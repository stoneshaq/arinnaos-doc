# Navigation

ArinnaOS is organised around operational workflows.

## Main Areas

- Dashboard
- CRM
- Suppliers
- Tours
- Passengers
- Quotes & Bookings
- Finance
- Settings

## Search

Use the command palette to quickly navigate across modules.

Shortcut:

- Cmd + K (macOS)
- Ctrl + K (Windows)
