# Navigation

ArinnaOS is organised around operational areas.

## Main areas

- Dashboard
- CRM
- Suppliers
- Tours
- Operations
- Passengers
- Quotes & Bookings
- Finance
- Settings

## Global search

ArinnaOS is designed to support a command palette for quick navigation.

Shortcut:

- Cmd + K on macOS
- Ctrl + K on Windows
