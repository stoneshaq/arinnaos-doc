# Invite-only Access

ArinnaOS does not allow public registration.

Access is controlled by invitations.

## Organisation creation

Platform admins create organisations.

After an organisation is created, a platform admin can invite the first organisation admin.

## User invitations

Organisation admins can invite users into their organisation.

The invitation defines:

- Email address
- Organisation
- Roles
- Expiry
- Invitation status

## Accepting an invitation

When a user accepts an invitation, ArinnaOS validates the token, checks the email address, creates or links the user, activates membership, and assigns approved roles.

## Invitation statuses

Invitations may be:

- Pending
- Accepted
- Expired
- Revoked

Expired or revoked invitations cannot be accepted.
