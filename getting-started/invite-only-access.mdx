# Invite-only Access

ArinnaOS does not support public registration.

Users can only access an organisation through invitations.

## Why Invite-only?

Invite-only access provides:

- Better security
- Better tenant isolation
- Controlled onboarding
- Reduced spam accounts

## Invitation Workflow

1. Organisation admin sends invitation.
2. User receives invitation.
3. User accepts invitation.
4. Membership is created.
5. Roles are assigned.

## Email Matching

Invitations are tied to an email address.

The invited email must match during acceptance.
