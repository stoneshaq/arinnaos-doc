# Invite-only Access

ArinnaOS does not allow public user registration.

All access begins with an invitation.

## Organisation onboarding

A platform admin creates an organisation and may invite the first organisation admin.

## User invitations

Organisation admins can invite users to their organisation.

An invitation includes:

- Organisation
- Email address
- Invited roles
- Token
- Expiry date
- Status

## Invitation acceptance

When an invitation is accepted, ArinnaOS:

1. Validates the token
2. Checks the invitation email address
3. Creates or links the user
4. Marks the email as verified
5. Creates or reactivates membership
6. Assigns validated roles

## Invitation statuses

Invitations can be:

- Pending
- Accepted
- Expired
- Revoked

Expired and revoked invitations cannot be accepted.
