# Invite-only Access

ArinnaOS is invite-only.

There is no public registration and no self-service organisation signup.

## Invitation workflow

1. A platform admin creates an organisation.
2. An organisation admin is invited.
3. The invited user accepts the invitation.
4. Membership is created or reactivated.
5. Roles are assigned.
6. The user enters the organisation workspace.

## Email matching

Invitations are tied to an email address.

The invited email must match when accepting the invitation.

## Expired and revoked invitations

Expired or revoked invitations cannot be accepted.
