# Users and Roles

Users access ArinnaOS through memberships.

## User

A user account is a person.

Examples:

- Travel consultant
- Operations manager
- Accountant

## Membership

Membership connects a user to an organisation.

A user may belong to multiple organisations.

## Roles

Roles define responsibility.

Examples:

- Organisation Admin
- Sales
- Operations
- Finance

Roles can be combined where appropriate.
