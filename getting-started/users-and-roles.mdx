# Users and Roles

Users access ArinnaOS through organisation memberships.

## Users

A user account represents a person.

## Memberships

A membership connects that person to an organisation.

## Roles

Roles group permissions for common responsibilities.

Examples:

- Organisation admin
- Sales
- Operations
- Finance
- Support

## Permissions

Permissions control specific actions such as viewing customers, managing quotes, converting bookings, issuing invoices, or allocating payments.
