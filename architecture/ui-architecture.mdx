# UI Architecture

ArinnaOS uses a custom dark-first UI foundation.

## Stack

- Laravel 13
- Livewire 4
- Tailwind CSS 4
- Alpine.js
- Tabler Icons
- TallStackUI selectively

## Design direction

- Dark-first
- Black and charcoal surfaces
- Gold accent
- Premium B2B SaaS feel
- Shopify Admin structure
- Spaceship dark UI polish
- Linear and Stripe clarity

## Component namespace

Custom components live under:

`x-arinna.*`

TallStackUI is used for selected interactions, not the full visual identity.
