# System Overview

ArinnaOS is a Laravel-based, multi-tenant SaaS platform for travel businesses.

## Main domains

- Platform administration
- SaaS billing
- Organisation and users
- CRM
- Suppliers
- References
- Tours and departures
- Passengers
- Quotes
- Bookings
- Travel finance
- Documents, notes, tasks, notifications
- Scheduler and command infrastructure
- UI foundation

## Architectural principles

- Organisation-scoped data isolation
- Permission-based access
- Snapshot-based commercial records
- Explicit conversion boundaries
- Travel finance separated from SaaS finance
- Invite-only onboarding
- Service/action business logic
- Activity logging from the start
